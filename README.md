# Iowa Gambling Task: Exploratory Analysis

This project explores behavioral trends in the **100-trial Iowa Gambling Task (IGT)** using publicly available data from the [Open Science Framework](https://osf.io/5pfey). The IGT simulates real-life decision-making under uncertainty, where participants choose from four decks (Aâ€“D) with different reward/loss profiles.

## Key Analyses

- **Deck Preference Over Time**: Tracks how choices shift across 5 trial blocks.
- **Net Score Calculation**: Measures decision quality via `(C + D) - (A + B)`.
- **Reward/Loss Summary**: Confirms payoff structure per deck.
